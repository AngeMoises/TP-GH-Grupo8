Ejecución de Tests
Este proyecto utiliza JUnit 5 para realizar pruebas automáticas sobre los componentes del sistema.

La clase SuiteCompleta agrupa y ejecuta automáticamente los siguientes tests:
    2.1. ThresholdAlertServiceTest
    2.2 AlertObserverFakeTest
    2.3AlertObserverMockTest

Para ejecutar los tests es necesario tener:
    Java instalado
    JUnit 5 configurado en el proyecto

Si todos los tests se ejecutan correctamente:
    JUnit mostrará los tests en color verde
    Se indicará que todas las pruebas pasaron exitosamente
